# Rebuild - Hebrew Daily Fitness Dashboard (RTL) ## Project Overview Rebuild is a comprehensive daily dashboard for athletes, focusing on Readiness, Fatigue, and Fitness. It integrates workout tracking, nutrition, and daily journaling into a single, cohesive RTL interface. ## Core Features - **Daily Dashboard:** Readiness/Fatigue/Fitness scores, "What to do today" guidance. - **Workout Integration:** Auto-sync with Strava and other services. - **Workout Log & Details:** Activity history, detailed views (Map, HR chart, splits/segments, feedback). - **Integrated Day Journal:** Morning check-in, nutrition, hydration, and target status on the 'Today' page. - **Nutrition System:** Macro tracking vs. targets, Pantry, Favorites, Quick Add. - **Hydration:** One-tap water logging. ## Technical Context - **Platforms:** Online web app (Vercel + Postgres), iOS app prototype for daily view/logging. - **Language/Layout:** Hebrew (RTL). ## Design Objectives - Create a high-performance, data-rich but intuitive interface. - Ensure full RTL support for Hebrew typography and layout. - Seamlessly blend health metrics with actionable daily tasks.
